# 属性

| 参数  | 说明           | 类型            | 默认值 |
| ----- | :------------- | :-------------- | :----: |
| name  | 名称           | string          |   -    |
| size  | 尺寸           | string / number |   16   |
| color | 颜色           | string          |   -    |
| spin  | 是否有旋转动画 | boolean         | false  |
| cache | 是否缓存图标   | boolean         |  true  |
